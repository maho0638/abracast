# Contributing

Abracast welcomes small, practical improvements that help Abstract users stay safe and organize their activity.

## Good first contributions

- Add official Abstract resource links
- Improve the safety scanner wording
- Add Discover app templates
- Improve mobile layout
- Add Turkish translations
- Add screenshots to the README

## Contribution rules

- Do not add private-key or seed-phrase inputs.
- Do not add token claim flows.
- Do not add hidden tracking.
- Keep the app useful without a backend where possible.
- Prefer clear user safety over hype.

## Pull request checklist

- Explain what changed
- Explain why it helps users
- Include screenshots for UI changes
- Verify that `index.html` still opens directly in a browser
