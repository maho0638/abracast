# Builder Role Packet

Use this when sharing Abracast with Abstract moderators, developer channels, or ecosystem reviewers.

## Short pitch

Abracast is an open-source ecosystem radar for Abstract. It helps users, streamers, and builders identify which Abstract projects are worth playing, streaming, funding, reviewing, or building around.

## Why it matters

Abstract's consumer chain thesis depends on useful apps, streamable games, creator activity, trading, marketplace usage, mints, tips, and retention. Portal discovery and XP incentives can send users toward projects, but users still need quality intelligence.

Abracast fills that gap by scoring projects on:

- User value
- Economic activity
- Replayability
- Streamer potential
- Onboarding quality
- Safety and trust
- Update cadence
- Abstract fit

Security is a gate, not a footnote. A project cannot receive a Support label unless it also has enough trust evidence: official source trail, wallet safety, economic clarity, verification/audit evidence, and transparent communication.

Abracast separates source confidence from hype. Directory listings help discovery, but a Support label requires stronger evidence such as official docs/site plus reliable ecosystem or third-party confirmation.

## What Abracast does

- Ranks projects as Support / Watch / Avoid
- Produces weekly Abstract picks
- Gives streamers content missions
- Shows repeated project gaps
- Helps builders understand what to fix
- Exports a shareable ecosystem radar report
